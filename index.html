MarketAssistant

一个开源的股票市场助手，提供实时股市信息、股票详情分析，支持MCP配置和本地向量内容搜索

主要特性

实时获取全球主要股市数据，包括A股、美股、港股等市场的实时价格、涨跌幅、成交量等关键指标

提供多维度的股票分析，包括K线图、技术指标（MACD、RSI等）、财务数据、新闻舆情等深度分析

MCP（Model Configuration Parameters）模型配置参数管理，通过配置文件灵活调整模型行为

基于FAISS向量数据库的本地搜索能力，快速检索股票相关的新闻、公告、研报等内容

功能详情

股市信息展示

支持全球主要股票市场，包括A股、美股、港股、日股等，展示实时价格、涨跌幅、成交量、换手率等核心数据，并提供图表可视化展示。

股票详情分析

提供全面的股票分析工具，包含：

  • K线图与技术指标分析（MACD、RSI、均线系统等）
  • 财务数据分析（营收、利润、资产负债等核心指标）
  • 新闻舆情分析（正负面情绪识别、热点事件追踪）
  • 历史数据回溯与对比分析

MCP配置

MCP（Model Configuration Parameters）是X2Agent的模型配置参数系统，允许用户通过YAML配置文件调整模型的行为特征。支持的配置项包括：

  • 数据更新频率与时间范围设置
  • 风险偏好参数调节
  • 算法模型选择（不同预测模型切换）
  • 个性化指标权重配置

本地向量搜索

X2Agent采用FAISS向量数据库实现高效的本地搜索功能，支持对以下内容进行快速检索：

  • 股票相关新闻（标题、正文、来源等）
  • 公司公告与财报
  • 行业研究报告
  • 社交媒体讨论内容
搜索响应时间低于200ms，支持自然语言查询和相关性排序。

使用场景

X2Agent为各类用户提供实用价值：

  • 投资者：快速获取股票全景信息，辅助投资决策
  • 分析师：整合多源数据，提升研究效率
  • 金融学生：学习股票分析方法和实践工具
  • 量化交易者：获取标准化数据接口，开发交易策略
